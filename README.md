# SmartHouse_Board
This repo is a part of a "Smart Home" Project for the course "AOS" of Politecnico 17/18 The project consists of 2 repo:
* SmartHouse_App: The client that the home owner uses to communicate with the board to modify the LEDs, Read temp and …
* SmartHouse_Board: The code based on Miosix to run on a STM board that will operate the user requests to the hardware like LEDs, and sends information like Temp. Politecnico di Milano
